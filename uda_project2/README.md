## SETUP DATABASE
1. Download postgres: set username = postgres and password = postgres
2. Create database with name = project2 for dev environment and name = project_test for test environment
3. migrate data with "db-migrate up"

## PORTS
1. Backend port: 3000
2. DB port: 5432

## STEPS:
- install: npm i
- run: npm start
- test: npm test
- buid: npm build
